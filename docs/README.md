supporting project records.

| `BUILD_WEEK_REQUIREMENTS.md` | Submission checklist | Active |

| `../STAGE_TRACKER.md` | Current stage and bounded slice tracker | Active tracker |
| `../PROJECT_STATUS.md` | Current project snapshot and blockers | Active snapshot |
| `../DECISION_LOG.md` | Binding project decisions | Active log |

The old source-package copy is preserved as
`../_archive/Code_Garden_Source_Package_v1/Code_Garden_Codex_Package_v1/docs/FROZEN_BASELINE.md`.

from the blueprint but follow the root tracker and record any material change in the root decision
log.
